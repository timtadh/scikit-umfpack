scikits.umfpack.UmfpackLU.perm_r¶

UmfpackLU.perm_r¶

Permutation Pr represented as an array of indices.

The row permutation matrix can be reconstructed via:

>>> Pr = np.zeros((n, n))
>>> Pr[perm_r, np.arange(n)] = 1
